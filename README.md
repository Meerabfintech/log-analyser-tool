# Log Analyzer Tool 

A Python tool that analyzes log files using regular expressions, finds errors and warnings, and generates reports. Built as part of Google IT Automation Course 2.

## Skills Demonstrated

- **Regular Expressions**: Pattern matching for errors, warnings, and IP addresses
- **Error Finding**: Detects and categorizes different types of errors
- **Testing**: Unit tests to ensure code reliability
- **Bash Scripting**: Shell script to automate the analysis process
- **File Handling**: Reads log files and writes reports
- **Object-Oriented Programming**: Clean class structure

## Features

-  Read and analyze any log file
-  Find errors using regex patterns
-  Find warnings in log files
-  Extract IP addresses
-  Count different error types
-  Generate summary reports
-  Unit tests for reliability
-  Bash launcher script

## Requirements

- Python 3.6 or higher
- No external libraries needed!
